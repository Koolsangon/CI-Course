        COP 원가 구조
        핵심 공식

        COP = COM(제조원가) + SG&A(판관비)

        COM = 소요재료비 + 가공비

        소요재료비 = Σ(공정별 BOM / 해당공정~최종 누적수율)

        가공비 = Panel(노무비+경비+상각비) + Module(노무비+경비+상각비)

        영업이익 = Price - COP

        Cash Cost = COP - 상각비

        EBITDA = Price - Cash Cost

        변동비 = 재료비 + 외주비 + 운반비

        한계이익 = Price - 변동비

        6가지 시뮬레이션 핵심 포인트

        1. Loading: 가동률 ↓ → 단위당 가공비 ↑ (고정비 배분)

        2. 인건비: 노무비 직접 증가 + SGA 인건비 비중도 연동

        3. 한계이익률: 목표율에서 역산하여 허용 변동비 도출

        4. 재료비/수율: 재료비 절감 vs 수율 하락 trade-off

        5. 면취수/Mask: 면취수 ↑ → Panel 비용 ↓, Mask ↑ → Panel 비용 ↑

        6. T/T+투자: Tact 지연 → Module 비용 ↑, 투자 → 상각비 ↑
